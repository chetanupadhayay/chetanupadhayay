### 01 · 🛒 [Olist E-Commerce Analytics](https://github.com/chetanupadhayay/Olist-Ecommerce-Analytics) *(Flagship Project)*

> **MySQL + Power BI** &nbsp;·&nbsp; 100K+ orders &nbsp;·&nbsp; 9 tables &nbsp;·&nbsp; 2016–2018

Brazil's largest e-commerce platform dataset — analyzed end-to-end from raw CSV files to an interactive Power BI dashboard. Built a complete MySQL database with 9 tables and foreign keys, then wrote 19 sections of SQL covering everything from basic EDA to advanced analytics.

- 🗄️ **Database:** Complete schema design with foreign keys + `LOAD DATA` import pipeline
- 🔍 **SQL Depth:** Window Functions (`RANK` · `DENSE_RANK` · `NTILE`), Stored Procedures, Views, CTEs
- 👥 **Customer Analytics:** RFM Segmentation · Cohort Retention · Customer Lifetime Value (CLV)
- 🚚 **Operations:** Delivery performance — on-time rate, avg delay days, late orders by state
- 📊 **Dashboard:** DAX measures, data modelling, relationships, interactive slicers in Power BI

```
📦 99,441 orders     💰 R$16M revenue      🚚 92.1% on-time
👤 96,096 customers  🔁 3.1% repeat rate   🏙️ SP = 38% revenue
```

### 02 · ✈️ [Airline Performance & Delay Analysis](https://github.com/chetanupadhayay/Airline-Performance-Delay-Analysis)

> **Power BI** &nbsp;·&nbsp; 6 Million flights &nbsp;·&nbsp; 2015

Analyzed the entire 2015 US domestic flight dataset to uncover delay patterns, cancellation root causes, and airline performance rankings. Built an interactive dashboard with month, airline, and cancellation type slicers.

- ✈️ **Performance:** Ranked all airlines by avg delay — identified best and worst performers
- ❌ **Cancellations:** Broke down cancellation reasons — Weather vs Airline vs Air System
- 📅 **Trends:** Monthly flight volume trend — peak and low traffic months identified
- 🛫 **Status Split:** On-Time vs Delayed vs Cancelled distribution across all flights

```
🛫 6M total flights    ✅ 81% on-time         ❌ 90K cancellations
📅 July peak (521K)    ⚠️ Spirit = 15.7 min   🌧️ Weather = 54.35% cancellations
```
